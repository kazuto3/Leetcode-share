### Leetcode199，二叉树的右视图

​		给定一个二叉树的 **根节点** `root`，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

![](D:\360download\Typora\ProcessOn\图片\二叉树的右视图.jpg)

思路：

（1）利用BFS进行层序遍历，记录下每层的最后一个元素

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public List<Integer> rightSideView(TreeNode root) {
        List<Integer> res = new ArrayList<>();
        if(root==null){
            return res;
        }
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);
        while(!queue.isEmpty()){
            int size = queue.size();
            for(int i=0;i<size;i++){
                TreeNode node = queue.poll();
                if(i==size-1){
                    res.add(node.val);
                }
                if(node.left!=null){
                    queue.offer(node.left);
                }
                if(node.right!=null){
                    queue.offer(node .right);
                }
            }
        }
        return res;
    }
}
```



（2）使用当前遍历的深度和结果集的大小作为限制，当当前深度还没有添加元素的时候，优先遍历最右边的元素并添加，添加后遍历本层其它元素并不会影响结果集中的值。

```java
class Solution {
    public List<Integer> rightSideView(TreeNode root) {
        List<Integer> res = new ArrayList<>();
        getRightNode(root,res,0);
        return res;
    }

    public void getRightNode(TreeNode root,List<Integer> res,int depth){
        if(root==null){
            return;
        }
        if(depth==res.size()){
            res.add(root.val);
        }
        getRightNode(root.right,res,depth+1);
        getRightNode(root.left,res,depth+1);
    }
}
```
